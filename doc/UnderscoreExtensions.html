static Class: UnderscoreExtensions

Public extensions to the underscore library

These extensions were written to assist development of the node-config library. They have been made public for other modules to use. To use these extensions, simply include the underscore library and call them as if they were a part of the library. Example: 
var _ = require('underscore');
_.extendDeep(extendInto, extendFrom, {some:"defaults"});

Methods

_.cloneDeep

object _.cloneDeep ( copyFrom , depth )
Return a deep copy of the specified object. This returns a new object with all elements copied from the specified object. Deep copies are made of objects and arrays so you can do anything with the returned object without affecting the input object.
Parameters:
copyFrom <object> The original object to copy from
depth <integer> An optional depth to prevent recursion. Default: 20.
Returns: object
A new object with the elements copied from the copyFrom object

_.extendDeep

object _.extendDeep ( mergeInto , mergeFrom... , depth )
Extend an object, and any object it contains. This does not replace deep objects like _.extend(), but dives into them replacing individual elements instead.
Parameters:
mergeInto <object> The object to merge into
mergeFrom... <object...> - Any number of objects to merge from
depth <integer> An optional depth to prevent recursion. Default: 20.
Returns: object
The altered mergeInto object is returned

_.isObject

boolean _.isObject ( arg )
Is the specified argument an object? The argument is an object if it's a JS object, but not an array and not a function.
Parameters:
arg <MIXED> An argument of any type.
Returns: boolean
TRUE if the arg is an object, FALSE if not

_.out

eyes.inspect _.out ( name , obj , deep , eyesConfig )
Print to the output stream This is a handy method for printing stuff to the output stream.
Parameters:
name <string> (optional) The object name
obj <object> The object to print
deep <boolean or integer> Print deep? Default: 2k output. TRUE prints up to 32k, or specify a size for the output.
eyesConfig <object> An optional eyes configuration object (see the eyes module)
Returns: eyes.inspect
Whatever eyes.inspect() returns

_.outStr

string _.outStr ( name , obj , deep , eyesConfig )
Return a string representation of an object This does the same thing as _.out(), except instead of sending the output to the standard output, the output is returned as a string.
Parameters:
name <string> (optional) The object name
obj <object> The object to print
deep <boolean or integer> Print deep? Default: 2k output. TRUE prints up to 32k, or specify a size for the output.
eyesConfig <object> An optional eyes configuration object (see the eyes module)
Returns: string
The formatted object as a string
